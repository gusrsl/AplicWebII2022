#### Explicacion del Avance:

Dados los lineamientos se trabajo en el microservicio de buses

1. Abrir una terminal con la ruta de esta practica complementaria es decir dentro de la carpeta microservicio
2. Abrir docker Desktop
3. Escribir en la consola los comandos "docker pull node" y "docker pull mongo" y "docker pull nginx" para que asi se instalen las imagenes necesarias
4. Una vez realizado lo anterior escribir en la terminal el comando "docker-compose up --build" para que asi se construya una imagen como tal de este proyecto y verificar si hay errores, se puede probar el funcionamiento
5. Como ultimo paso detener el proceso anterior y esribir en la terminal el comando "docker-compose up -d" para asi poner en marcha la aplicacion sin tener informacion de mas en la terminal
